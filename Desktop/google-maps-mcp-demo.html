Google Maps MCP Server Demo

Overview

This demo showcases the capabilities of the Google Maps MCP server, which provides tools for geocoding, reverse geocoding, place search, directions, and more. The server is integrated with Claude using the Model Context Protocol (MCP).

1. Geocoding

The maps_geocode tool converts an address into geographic coordinates (latitude and longitude).

Example: Sydney Opera House, Australia
{
  "location": {
    "lat": -33.8567844,
    "lng": 151.2152967
  },
  "formatted_address": "Bennelong Point, Sydney NSW 2000, Australia",
  "place_id": "ChIJ3S-JXmauEmsRUcIaWtf4MzE"
}

2. Reverse Geocoding

The maps_reverse_geocode tool converts geographic coordinates into an address.

Example: Coordinates of Sydney Opera House
{
  "formatted_address": "75 Castlereagh Street, Sydney NSW 2000, Australia",
  "place_id": "ChIJB5EajGauEmsRSv7kPQuhpls",
  "address_components": [
    {
      "long_name": "75",
      "short_name": "75",
      "types": [
        "street_number"
      ]
    },
    {
      "long_name": "Castlereagh Street",
      "short_name": "Castlereagh Street",
      "types": [
        "route"
      ]
    },
    ...
  ]
}

3. Place Search

The maps_search_places tool searches for places using a text query, optionally within a specified radius of a location.

Example: Restaurants near Sydney Opera House
{
  "places": [
    {
      "name": "Grana",
      "formatted_address": "Ground Floor/5-7 Young St, Sydney NSW 2000, Australia",
      "location": {
        "lat": -33.8625166,
        "lng": 151.2109494
      },
      "place_id": "ChIJvyZ8WvCvEmsRTqM2KE94_aw",
      "rating": 4.6,

4. Directions

The maps_directions tool provides directions between two locations, with options for different travel modes.

Example: Driving directions from Sydney Opera House to Bondi Beach
{
  "routes": [
    {
      "summary": "M1",
      "distance": {
        "text": "9.1 km",
        "value": 9066
      },
      "duration": {
        "text": "17 mins",
        "value": 1032
      },
      "steps": [
        {
          "instructions": "Head east on Macquarie St",
          "distance": {
            "text": "0.7 km",
            "value": 737
          },
          "duration": {
            "text": "3 mins",
            "value": 175
          },
          "travel_mode": "DRIVING"
        },
        ...
      ]
    }
  ]
}

How to Use

You can use these tools by asking Claude to perform geographic tasks. For example:

  • "What are the coordinates of the Eiffel Tower?"
  • "What's the address at these coordinates: 40.7128, -74.0060?"
  • "Find restaurants near Times Square, New York."
  • "How do I get from Central Park to the Empire State Building by walking?"
